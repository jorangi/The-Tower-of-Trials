### 정통 판타지: 무한의 탑 (The Tower of Trials)
- 가장 기본적이고 직관적인 RPG 구조입니다. 상태(State) 전환과 클래스 상속을 연습하기 완벽합니다.
- 배경: 대륙 중앙에 솟아오른 정체불명의 마탑. 층이 올라갈수록 강한 몬스터가 등장합니다.
- 스토리: 저주받은 마을을 구하기 위해 마탑 꼭대기에 있다는 '정화의 코어'를 찾으러 간 초보 모험가의 이야기입니다.
- 게임 흐름: [마을(회복 및 정비)] -> [탑 입장(층수 선택)] -> [전투] -> [보상 획득 및 다음 층 이동]

라는 느낌으로 만들게 되었다.

머드 게임이니까 딱히 그래픽 요소는 넣을 생각이 없다.<br>
나중에 Win API랑 Graphics 공부 좀 더 하고 간단한 프로젝트는 생각중이긴 한데, 이걸 기반으로 확장할지도 모르겠다.<br>

일단 이번 커밋에서 진행한 내용은
- StatModifier 추가
    - StatModifierType 열거형 추가
    - sourceId는 uint32_t로 했는데, 64_t는 너무 커서 불필요해보이고 32로 했다. int가 아닌 uint를 쓴 이유는 음수 아이디는 없을 예정이고, 눈에 보기 편하다.
    - C++은 끽해야 알고리즘 문제용으로 써보았던게 다였다 보니, 헤더 파일 쓰는게 익숙치 않았다.
- ModifiableStat 추가
    - StatModifier 구현부
    - STL을 활용한 기능 구현
        - STL은 학습할 때 조금 써봤는데, 확실히 단순 C랑 확실히 생산성 차이가 있는 것 같다.
        - var랑 비슷한 auto가 있는데, C#에서도 사실 var 안쓰고 직접 타입 입력하긴 했었는데, 확실히 편하긴 하다..
